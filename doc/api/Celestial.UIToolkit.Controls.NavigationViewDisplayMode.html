Enum NavigationViewDisplayMode

Defines the different display modes of a NavigationView control.

Namespace: Celestial.UIToolkit.Controls
Assembly: Celestial.UIToolkit.dll
Syntax
public enum NavigationViewDisplayMode

Fields

Name Description
Compact

The NavigationView shows a small strip which can be expanded via the menu button.
Expanded

The NavigationView is fully expanded, meaning that the pane stays open the whole time.
Minimal

The NavigationView displays only the minimal amount of elements, which happens to be the menu button. With this button, the pane can be shown and hidden.
