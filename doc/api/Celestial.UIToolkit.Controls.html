Namespace Celestial.UIToolkit.Controls

Classes

Card

A ContentControl which depicts a card. In addition to the the Content, the Card provides a set of additional content properties like Title and MediaContent to organize the content into structured parts. This allows control templates to change the arrangement of the card's different elements, while giving the user full control over what the content looks like.

ClippingBorder

An extension of the Border class which applies a clip to the underlying child, so that the child will not be rendered over the border's edges, if a corner radius is set.

CommonVisualStatesControl

An extension of the Control class, which implements common visual state changes.

ConstructedFrameworkElement

A special FrameworkElement which provides sub-classes with a "Child" property, similar to WPF's Decorator. This allows sub-classes to automatically build a fixed element tree, without the user being able to change it. Hence the term "Constructed", because sub-classes are supposed to construct the element tree.

DirectionalThemeShadow

A ThemeShadow which moves the shadow in a certain direction, defined by the ShadowDirection property.

DropShadowDecorator

A control which adds a drop shadow effect to its content. This control uses the standard WPF DropShadowEffect, but simplifies its usage via more intuitive properties.

ExtendedListView

An extension of the ListView control which provides events and methods dealing with its Item-Containers.

IconElement

The base class for an icon UI element.

ListViewItemInvokedEventArgs

Provides event data for when a ListViewItem is invoked.

ManipulationBoundaryFeedbackAwareContentControl

A ContentControl which reacts to the ManipulationBoundaryFeedback event of either the content, and/or an element directly specified via the FeedbackProvider property.

NavigationView

A control which provides a central navigation structure to an application by providing a pane for navigation commands and a central place for the content which is the current navigation target.

NavigationViewDisplayModeChangedEventArgs

Provides event data for the DisplayModeChanged event. This class cannot be inherited.

NavigationViewItem

Represents the container for a generic navigation item in a NavigationView.

NavigationViewItemEventArgs

Provides generic event data regarding a single item which is hosted by a NavigationView.

NavigationViewListView

A custom ListView which is supposed to be used with a NavigationView for displaying the MenuItems. In comparison to a normal ListView, this class provides events for when an item gets invoked, creates NavigationViewItem container elements by default and disallows Keyboard Navigation.

PathIcon

An IconElement which draws its shapes by using a Geometry.

PlaceholderOverlay

A control which displays placeholder content over the actual content.

RelativeCanvas

Defines an area within which you can position children based on coordinates, relative to the canvas element itself.

RippleOverlay

A content control which provides a ripple animation effect to the underlying content, whenever the user clicks on it. The effect is based on the ripple effect by the Material Design language.

SelectionIndicator

A basic control which is usually used within control templates. It indicates whether the hosting control is currently selected (or anything similar).

SplitView

Represents a container with two areas. One area is a pane which typically displays navigation elements or popup-content. The other area displays the main content which typically relates to the elements in the pane.

ThemeShadow

A special type of shadow which aids in creating a unified shadow experience across the whole application by providing an elevation API to create layering and depth effects.

It can be attached to any type of controls via the ShadowProperty. For the shadow to be displayed, control template authors are required to display the ThemeShadow provided via the attached ShadowProperty. The easiest way to do this is by simply adding a ThemeShadowPresenter to the control template.

ThemeShadowPresenter

A control which is used in conjunction with the ThemeShadow. When added to a control's template, it displays a ThemeShadow which is attached to an element via the ShadowProperty.

ToggleSwitch

Represents a switch which allows users to turn something on or off.

WindowCommands

Provides static ICommand implementations which can be used to add Minimize, Maximize, Restore and Close handlers to custom window caption buttons.
