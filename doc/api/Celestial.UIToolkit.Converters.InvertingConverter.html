Class InvertingConverter

An implementation of the IValueConverter interface which uses another IValueConverter specified via the UnderlyingConverter property to convert values, but in an inverted way.

In the Convert(Object, Type, Object, CultureInfo) method, the underlying converter's ConvertBack(Object, Type, Object, CultureInfo) method is called. Likewise, in the ConvertBack(Object, Type, Object, CultureInfo) method, the underlying converter's Convert(Object, Type, Object, CultureInfo) is called.
