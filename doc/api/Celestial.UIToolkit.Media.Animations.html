Namespace Celestial.UIToolkit.Media.Animations

Classes

AnimationBase<T>

Defines a base class for a custom AnimationTimeline, following the pattern in the .NET Framework (see remarks).

AnimationUsingKeyFramesBase<T, TKeyFrame, TKeyFrameCollection>

A base class for an animation using key frames.

BrushAnimation

Animates the value of a Brush property between two targets, using linear interpolation over a specified Duration.

BrushAnimationUsingKeyFrames

Animates the value of a Brush property along a set of KeyFrames.

BrushKeyFrameCollection

A freezable collection which is able to hold key frames of Brush objects.

DiscreteBrushKeyFrame

Animates from the Brush value of the previous key frame to its own Value using discrete interpolation.

DiscreteKeyFrameBase<T>

An abstract base class for a discrete key frame.

EasingBrushKeyFrame

A key frame which allows you to associate easing functions with a BrushAnimationUsingKeyFrames animation.

EasingFromToByAnimationBase<T>

Defines an abstract base class for a From/To/By animation, which, in comparison to the FromToByAnimationBase<T>, additionally uses an easing function.

EasingKeyFrameBase<T>

An abstract base class for a key frame which can be associated with an easing function.

FromToByAnimationBase<T>

Defines an abstract base class for a From/To/By animation.

KeyFrameBase<T>

An abstract base class for a key frame that participates in a key frame animation.

LinearBrushKeyFrame

Animates from the Brush value of the previous key frame to its own Value using linear interpolation.

SplineBrushKeyFrame

Animates from the Brush value of the previous key frame to its own Value using splined interpolation.

SplineKeyFrameBase<T>

An abstract base class for a key frame using splined interpolation.

Interfaces

IAnimationClock

Defines a clock which maintains the run-time state of an AnimationTimeline and processes its output values.

IClock

Defines a clock which maintains run-time timing state for a Timeline.

IClockController

Represents an object which can interactively control an IClock.

IInterpolateKeyFrame<T>

An extension of the IKeyFrame interface, which defines a key frame which has the ability to interpolate its value, depending on a progress factor.
