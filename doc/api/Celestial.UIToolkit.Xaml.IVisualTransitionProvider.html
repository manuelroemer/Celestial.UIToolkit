Interface IVisualTransitionProvider

Defines an element which can generate a visual transition for a specific type of Timeline. This interface is being used by the VisualTransitionProvider and ExtendedVisualStateManager to create visual transitions between animations in different visual states.

Namespace: Celestial.UIToolkit.Xaml
Assembly: Celestial.UIToolkit.Core.dll

CreateFromTransitionTimeline(Timeline, IEasingFunction)

Called when the ExtendedVisualStateManager transitions away from the element. The timeline which gets returned by this method is then used as a transitioning animation.

Timeline fromTimeline

The input timeline, for which a visual transition timeline should be generated. The VisualStateManager wants to transition away from this timeline.
IEasingFunction easingFunction

An easing function to be applied to the resulting timeline. Can be null.
Returns
Type Description
Timeline

A Timeline which displays a visual transition away from this element.
