Namespace Celestial.UIToolkit.Xaml

Classes

BindingProxy

A class which is used to allow bindings in places where they wouldn't normally be possible, by creating a proxy between the two parties.

EnumBindingSourceExtension

A custom markup extension which takes an enum's type and returns all of its members as a set of string values. These strings can, for instance, be used as a Binding source.

ExtendedVisualState

An extension of the VisualState class which represents the visual appearance of the control when it is in a specific state. This class extends the default VisualState with support for setters.

ExtendedVisualStateManager

A custom visual state manager which provides the same features, aswell as additional ones, as the default VisualStateManager.

GridUnitExtension

A MarkupExtension which is being used by the toolkit to align visual elements in a grid-like fashion. The idea is that each control uses margins, paddings, sizes, etc. which can be fit into grid cells. Thus, when positioned correctly, all controls will have a unified appearance.

This extension provides unified values, multiplied with the bounds of a single grid cell. See the example for details.

MarginHelper

Provides attached dependency properties regarding margins which can be helpful when creating layouts.

MultiStyleExtension

A custom markup extension which merges multiple styles into one single style.

SelfBindingExtension

An otherwise normal Binding whose RelativeSource property is set to Self.

TemplatedParentBindingExtension

An otherwise normal Binding whose RelativeSource property is set to TemplatedParent.

VisualTransitionProvider

A class which provides the ExtendedVisualStateManager with transition timelines between states.

Interfaces

IVisualTransitionProvider

Defines an element which can generate a visual transition for a specific type of Timeline. This interface is being used by the VisualTransitionProvider and ExtendedVisualStateManager to create visual transitions between animations in different visual states.
