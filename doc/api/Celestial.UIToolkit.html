Namespace Celestial.UIToolkit

Classes

DesignMode

Provides static members for determining whether the application is currently in the designer mode.

DesignTimeOnlyResourceDictionary

A special resource dictionary which only loads its content, if it is currently in the design mode. Otherwise, the loading process will simply be ignored.

ItemsSourceCollection

A collection which can retrieve its items from an items source, if specified. If not, it behaves like a normal collection to which any item can be added.

SharedResourceDictionary

An extension of the ResourceDictionary class which shadows the Source property. The difference to a normal resource dictionary is that when this class' Source property is set, the dictionary from that source gets loaded via the Celestial.UIToolkit.SharedResourceDictionaryManager, which allows caching of the resource dictionaries. The loaded resource dictionary set by the Source property is then accessible via the MergedDictionaries property.

TraceSources

Provides access to static TraceSource instances which are used by the toolkit to trace information. This class has the same intent as the PresentationTraceSources class, but is created separately, so that WPF's trace messages are left untouched.
